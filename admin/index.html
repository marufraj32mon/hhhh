Embed Code
Add the chatbot to your website
🔗 Widget Script Tag
Add this single line to your website's HTML — works on any page, CMS, or theme.
<script src="https://YOUR-VERCEL-URL.vercel.app/widget.js"></script>
Replace YOUR-VERCEL-URL with your actual Vercel deployment URL.
🚀 Deployment Steps
  1. Install Vercel CLI: npm i -g vercel
  2. In the project folder: vercel login
  3. Deploy: vercel --prod
  4. Set environment variables in Vercel dashboard
  5. Copy the deployment URL and paste it in the embed code above
  6. Paste the script tag into your website's <body>
